# 你不懂JS（系列丛书）

这是一套深入探讨 JavaScript 语言核心机制的系列丛书。本系列的第一版已经完成。

欢迎通过提交 PR 改进代码段，讲解等，来为此内容的质量做出贡献。虽然也欢迎拼写错误的修改，但是它们很可能将在一般的编辑过程中被找出来，因此对本代码库不一定很重要。

要了解更多关于这个系列丛书背后的动机和角度，请参阅前言。
